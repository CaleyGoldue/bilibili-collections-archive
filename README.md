## 恭喜
发现了隐藏分支，收集act_id只是表象；<br>
该分支是尽可能多的备份收藏集里的表情包文件，动图默认只存储GIF版。<br>
但是分支内容请勿张扬宣传，容易导致项目被DMCA。
### 项目诞生真正的原因：
因有部分收藏集，奖池或者兑换物有时限，甚至卡面有数量限制；达到条件后只有获取过的用户可以查看到，未登录或者未获取的用户无法查看，甚至从正常接口都拉取不到。<br><br>
